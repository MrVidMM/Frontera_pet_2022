IMPORTANTE
-----------------

Esta es la guía de instalación de la web "FonteraPets"

Empezemos!!

Requisitos:

- Xampp instalado / https://www.apachefriends.org/es/index.html
- Node JS Instalado / https://nodejs.org/es/

1. Una vez descargado los archivos, extrae los archivos que se encuentran dentro del rar/zip

2. Abre xamp y inicia los servicios de "Apache" y "MySql"

3. En la fila de MySql pulsa "Admin"

4. Pulsa el botón "Importar". Importa la base de datos que esta en la carpeta "#Base_de_Datos" que se llama "db_api_app"

5. Ya tendrias la base de datos montada

6. Si ya tienes instalado NodeJS deberias tener los comandos que utilizaremos 

7. Abre tu editor de codigo (de preferencia "Visual Studio Code") o consola de comandos

8. Si estas en tu editor de codigo abre la carpeta "Frontera-dj-main"

8.1 Si estas en consola de comandos buscar la ruta de la carpeta "Frontera-dj-main"

9. En la carpeta "Frontera-dj-main" tendras un archivo llamado "requirements.txt"...

10. Deberas instalar cada uno de ellos con el comando: npm install <nombre>

11. Una vez terminado ejecuta el proyecto con el comando: python manage.py runserver

12. Listo!! Ya tendrias el proyecto en funcionamiento
